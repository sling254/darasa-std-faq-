Darasa FAQ

Darasa Requirements?


You should be online.
Location should be on.

How do i confirm class attendance?



To confirm class attedance click the button Add new attendance. and tap START SCAN to initiate scanning for lecturer inside the classroom.

How to set up Current Semester?


Click on the menu icon
select settings then click on Set Current Semester Make sure you have keyed in the Semester, Study year and Academic year. Click SAVE to effect changes.

How to Manage Accout?


Click on the menu icon
select settings then click on Manage Account Make sure you have keyed in your First name, Last name, Registration number, course, school and department. Click SAVE to effect changes.

How to check for class?


click Calendar icon on the right side of the screen and select the date you wish to check class..
